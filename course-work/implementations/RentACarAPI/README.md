Име: Антония Веселинова Глухова

Факултетен номер: 2401321003

## Описание

Уеб система за отдаване на автомобили под наем. Поддържа регистрация, вход, управление на автомобили, резервации, наеми, плащания и любими автомобили.

## Backend

ASP.NET Core Web API
Entity Framework Core
SQL Server
JWT Authentication

## Frontend

React
TypeScript
Vite

## Инсталация

1. Настройте връзката към базата данни в appsettings.json
2. Изпълнете миграциите:
   dotnet ef database update
3. Стартирайте API:
   dotnet run
4. Стартирайте Frontend:
   npm install
   npm run dev
